Informations sur la sauvegarde

Fréquence recommandée

Il est recommandé de créer des sauvegardes régulières :

  • Quotidienne : Pour les données critiques
  • Hebdomadaire : Pour les données importantes
  • Mensuelle : Pour les archives

Sécurité

Les sauvegardes contiennent toutes les données de l'application :

  • Base de données complète
  • Fichiers uploadés
  • Configuration système

Attention

Points importants à retenir :

  • La restauration peut écraser les données actuelles
  • Vérifiez toujours le contenu avant restauration
  • Conservez les sauvegardes dans un endroit sûr
